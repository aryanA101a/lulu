# lulu
A LC3 architecture based virtual machine written in GO.


## Build
`go build`


## Usage
1. **Flags**
```
  -h	help
  -logfile string
    	path of logfile
  -v	logging
  ```

2. **Examples**  
  `./lulu program.obj`       
  `./lulu -v program.bin`
